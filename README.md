# Calculadora de IMC

Este é um simples projeto em Ruby para calcular o Índice de Massa Corporal (IMC) e fornecer uma classificação com base no valor obtido.

## Requisitos

- Ruby 2.x ou superior

## Instalação

Este projeto não requer dependências adicionais. Certifique-se de ter o Ruby instalado em sua máquina.

## Uso

1. Clone o repositório:
   ```bash
   git clone https://github.com/usuario/calculadora-imc.git
   ```

2. Navegue até o diretório do projeto:
  ```bash
   cd calculadora-imc
  ```

3. Execute o script Ruby:
  ```bash
   ruby imc_calculator.rb
  ```
